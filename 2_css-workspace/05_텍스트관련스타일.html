텍스트 관련 스타일

color : 텍스트의 색상을 지정

        선택자{
            color : 색상명/16진수/rbg(x,x,x)/rgba(x,x,x,x) 등등;
        }

text-decoration : 텍스트에 줄을 긋거나, 없앨때 사용 

        선택자{
            text-decoration 
            : none(사용안하겟다) / underline(글자에 밑줄그어줌)
              / overline / line-through (취소선그어줌);
        }
  • underline : 텍스트영역 아래에 줄긋기
  • line-through : 텍스트영역 중간에 줄긋기
  • overline : 텍스트영역 위에 줄긋기
네이버로 이동

text-transform : 영문텍스트의 대소문자 변환시 사용

        선택자{
            text-transform : uppercase / lowercase / capitalize ;
        }
  • uppercase : 모든 영문자를 대문자로
  • LOWERcase : 모든 영문자를 소문자로
  • cAPitalize (hi) : 단어의 첫글자만 대문자로

text-align : 텍스트를 정렬할때 사용

        선택자{
            text-align : left(기본값) / justify / right / center;
        }

text-shadow : 텍스트에 그림자 효과를 줄때 사용

        선택자{
            text-shadow : 가로거리 세로거리 번짐정도 색상;
        }
        - 번짐정도 색상은 생략 가능
        - 가로거리와 세로거리가 양수일경우 => 오른쪽 아래에 그림자 생성
        - 가로거리와 세로거리가 음수일경우 => 왼쪽 위에 그림자 생성
        - 번짐정도가 0일 경우, 그림자 색이 제일 진함
        - 색상은 기본값으로 글자색과 동일
