1. 속성 선택자

스타일을 부여하고자하는 요소내에 작성되어있는 속성을 이용해서 선택하는 방법 선택자 뒤에 []를 이용해서 속성과 속성값을 제시해서 선택하는 방법 

        선택자[속성=속성값] -> 일치하는 요소를 선택
        
        선택자[속성~=속성값] -> "포함"되어있는 요소를 선택
        
        선택자[속성|=속성값] -> "시작"하는 요소(-앞에 한단어로)를 선택
        /* div - div */
        
        선택자[속성^=속성값] -> "시작"되는 요소를 선택 ( |보다 많이 사용 )
        
        선택자[속성$=속성값] -> 속성값으로 "끝"나는 요소를 선택

        선택자[속성*=속성값]{ -> 위치가 어디가 됏건, i를 "포함"하는 요소를 선택

            스타일속성 : 값;

        }

3. 동위선택자(같은레벨)

동위 관계(같은 레벨)에 있는 뒤에 위치한 특정요소를 선택할때 사용 

        * a요소 뒤에 있는 b요소 "하나만"을 선택
        a+b{
            스타일속성 : 값;
        }

        * a요소 뒤에 있는 "모든 b요소 다" 선택
        a~b{
            스타일속성 : 값;
        }

4. 반응선택자

사용자의 움직임에 따라 선택이 되거나 안되는 선택자 

        * 해당요소에 클릭이 되었을 경우만 스타일을 부여함
        선택자:active {
            스타일속성 : 값;
        }
        
        * 해당요소에 마우스가 올라갔을 경우만 스타일을 부여함
        선택자:hover { --> hover : 맴돌다라는 의미
            스타일속성 : 값;
        }
