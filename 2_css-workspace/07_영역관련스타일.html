기준 div

기준 div

1) padding

내용물 영역(content)과 테두리 영역(border) 사이에 거리(여백)를 조정할 수 있다.

컨텐츠 영역
-> padding : 100px;

컨텐츠 영역
-> padding-top : 50px; padding-left : 100px;

컨텐츠 영역
-> padding : 50px 20px 10px 0px;

2) margin

타 요소들 간의 간격을 조정할때 사용

컨텐츠 영역
margin:50px;

컨텐츠 영역
margin-top:50px; margin-left:20px;

컨텐츠 영역
margin : 50px 30px 20px 0px;
