목록 관련 스타일 (list~)

list-style-type : 불릿기호를 변경해줄때 사용

        * 순서없는 목록 (ul)
        선택자 {
            list-style-type : disc / circle / square / none(기본값) ;
        }

        * 순서있는 목록 (ol)
        선택자 {
            list-style-type : decimal (정수) / decimal-leading-zero (앞에 0추가)
            / lower-alpha (알파벳 소문자부터) / upper-alpha (알파벳 대문자부터) 
            / lower-roman (로만/ upper-roman ;
        }

순서없는 목록

  • HTML
  • CSS
  • JAVASCRIPT
  • JQUERY

순서있는 목록

  1. HTML
  2. CSS
  3. JAVASCRIPT
  4. JQUERY

list-style-position : 불릿기호의 위치를 조정할때 사용

list-style-image : 불릿기호로 이미지를 적용시키고자 할때 사용

        선택자{
            list-style-position : inside / outside ;

            list-style-image : url (적용시키고자 하는 이미지의 경로);
        }
